Claude Code ハンズオン開催のお知らせ

開催日時: 12/05(金) 17:30-18:30
形式: **オンライン開催**

---

ハンズオン内容

このハンズオンでは、最新の Web 開発技術を組み合わせた実践的なプロジェクトを通じて、Claude Code を使った効率的な開発方法を学びます。

学習できること：
・Claude Code の基本操作と機能（コード分析、生成、編集、プランモード）
・5つのセッションで段階的に習得（詳細は下記参照）

セッション内容：
👉 https://github.com/LDJP-AI/ai-workshop-series-claudecode-01/tree/main/sessions

※既にClaudeCodeを使ったことある人は知ってることが多いかも

---

この勉強会を通じて得られること

このハンズオンは **参加者が手を動かしながら学ぶ** 形式です：

・実装を通じた実践的な習得（「読む」ではなく「やる」）
・ClaudeCode の実機操作体験
・問題が発生した際の対処方法を学べる
・講師によるリアルタイムサポート（質問対応・ガイダンス）

**講師の役割：** 問題解決やガイダンスをサポートします。
参加者が主体的に進め、詰まったときにサポートする形式です。

---

対象者：Claude Code 初心者〜初級者向け

- 必要なツールのインストール確認

**最も重要：** 本ハンズオン前に、上記の README の手順に従い、アプリケーションが localhost:3000 で正常に起動することを確認してください。
