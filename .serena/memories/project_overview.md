# Project Overview

## Project Name

ai-workshop-series-claudecode-01

## Purpose

A Next.js starter project for the AI Workshop Series focused on Claude and modern web development. It serves as a template for learning and building web applications with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16.0.3 with App Router
- **Language:** TypeScript 5
- **UI Library:** React 19.2.0 with React Server Components
- **Styling:** Tailwind CSS 4
- **Build Tool:** Next.js CLI (webpack under the hood)
- **Linting:** ESLint 9 with Next.js configuration
- **Node Version Management:** Node.js (use with npm)

## Project Type

Frontend web application - a modern, server-side rendered React application with static site generation capabilities

## Key Dependencies

- next: 16.0.3
- react: 19.2.0
- react-dom: 19.2.0
- tailwindcss: 4
- eslint: 9 with eslint-config-next

## Deployment Target

Optimized for Vercel deployment (standard Next.js configuration)
